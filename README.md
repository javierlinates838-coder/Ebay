# Ebay

This repository contains **ResellAI** — an AI-powered eBay Reseller Assistant.

## ResellAI

See [`ebay-reseller/README.md`](ebay-reseller/README.md) for setup, environment variables, and deployment instructions.

```bash
cd ebay-reseller
npm install
cp .env.example .env.local
npm run dev
```

The app runs in demo mode without API keys configured.
